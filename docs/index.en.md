# Fri3d Camp 2026 Badge

This site contains the documentation for the Fri3d Camp 2026 badge.

The documentation setup mirrors the `badge_2024` project:

- MkDocs with the Material theme
- GitHub Pages deployment through GitHub Actions
- English and Dutch pages using `mkdocs-static-i18n`

## Status

The documentation structure is in place. Content for the badge, addons, onboarding, and programming guides can be added incrementally from here.
