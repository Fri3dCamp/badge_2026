# SaO Mirror

Deze printplaat is ontworpen om sensoren die je op de [SaO connector](https://hackaday.io/project/175182-simple-add-ons-sao) kan plaatsen, naar de "andere" kant van de badge te laten wijzen. Dit is vooral heel praktisch bij het gebruik van de [ToF addon](../tof). Als extra functie werden er ook 8 [WS2812 RGB LEDs](## "Neopixel LEDs") geplaatst voor extra visualisatie.
