# Fri3d Camp 2026 Badge

Deze site bevat de documentatie voor de Fri3d Camp 2026 badge.

De documentatie-opzet volgt hetzelfde patroon als `badge_2024`:

- MkDocs met het Material-thema
- GitHub Pages publicatie via GitHub Actions
- Engelse en Nederlandse pagina's via `mkdocs-static-i18n`

## Status

De documentatiestructuur staat klaar. Inhoud voor de badge, addons, onboarding en programmeergidsen kan vanaf hier stapsgewijs toegevoegd worden.

## Ontwikkeling

Zie de [README van de repository](https://github.com/fri3dcamp/badge_2026) voor instructies om de site lokaal te bekijken.
